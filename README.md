### 项目介绍
此项目是在此前作者本人开发的商业项目《秀场直播》中提取并在此基础上优化的以礼物模块为核心的功能交互模块。功能包含但不限于：在线直播间、完整礼物模块、礼物打赏交互、中奖、飘屏动画、弹幕、聊天互动等功能。欢迎下载体验APK！
项目中直播间控制器预留了IM消息接入，请阅读RoomBaseController类源码注释接入IM实现真实互动。

### 功能概述
* 在线直播间观看，支持直播间秒开</br>
* 完整的礼物模块，包括但不限于礼物动画、交互、选择、回显、赠送等。</br>
* 直播间礼物互动、中奖、飘屏弹幕</br>
* 直播间聊天互动示例</br>
* 直播间控制器层预留了IM消息入口</br>
